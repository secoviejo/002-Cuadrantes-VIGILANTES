# Fuentes documentales

Esta carpeta conserva documentos operativos externos que sirven como base de conocimiento del proyecto.

- `PTT-Vigilancia-UZ.md`: pliego tecnico de referencia para servicios de vigilancia y auxiliares de conserjeria de la Universidad de Zaragoza. Sus reglas se usan como fuente funcional para interpretar dias normales, festivos y no lectivos, y para orientar la futura generacion dinamica de cuadrantes.

Cuando llegue un nuevo PTT, debe archivarse aqui, compararse con la version anterior y documentar los cambios de reglas antes de trasladarlos a base de datos o servicios backend.
